Dream Book Store
A simple and visually appealing Creative Book Store Website built using HTML, CSS, and JavaScript. This project showcases a modern UI with interactive features like search, animations, and user actions.
✨ Features
🔍 Search Functionality – Filter books instantly
🎨 Modern UI Design – Glassmorphism effect with background image
📱 Responsive Layout – Works on different screen sizes
❤️ Like Button – Add books to favorites (alert-based)
🛒 Cart Button – Simulated add-to-cart feature
🌟 Hover Effects – Smooth animations on book cards
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Google Fonts (Poppins)
Unsplash Images
🚀 How to Run the Project
Download the project folder
Open the folder
Double-click index.html
OR open it using any web browser
🎯 How It Works
Users can search books using the search bar
Book cards are filtered dynamically using JavaScript
Clicking:
❤️ shows "Added to Favorites"
🛒 shows "Added to Cart"
